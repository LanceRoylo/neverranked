# Never Ranked. Voice Quick Reference

Distilled from the Hello Momentum tastemaking pipeline at
`/Users/lanceroylo/Desktop/AI-Agency/hello-momentum-agency/` plus
the Never Ranked-specific patterns established on the live site.

## The voice in one paragraph

Editorial, evidence-led, unsentimental. Reads like a Financial
Times feature, not a SaaS landing page. Specific numbers over
abstract claims. The italic Playfair mid-sentence treatment is
the signature distinctive asset. Tight. Cut every word that does
not earn its place.

## Banned mechanically

- Em dashes (use periods or commas instead)
- Semicolons in marketing copy (split into two sentences)
- Emojis on website pages, LinkedIn, X, Threads, Facebook
- Words: transform, unlock, empower, elevate, leverage, seamless,
  cutting-edge, dominate, Welcome to, We help you, Era (when it
  modifies a category), pristine, robust, holistic, synergy

## Banned by intent (subtler)

- Hidden gem, rare opportunity, game-changer, revolutionary
- Hyperbole without specificity ("the best", "the only" without
  receipts to back it)
- Call-to-action language that begs ("Don't miss out", "Last chance")
- "We're excited to..." openers
- Generic "AI" buzzword pitching when "ChatGPT" or "Perplexity" is
  more specific

## Specificity tests every post must pass

- **Swap test.** Replace "Never Ranked" with "any AI search agency."
  Does the post still make sense? If yes, it is too generic. Add
  the named engine, the named prompt, the actual number, the
  specific page on the customer's site.
- **So-what test.** Read the headline aloud. Is the audience's
  honest reaction "so what"? If yes, find the tension first.
- **Lever test.** Name the creative lever the post is using
  (Reframe, Specificity, Confession, Tension Hold, Format Surprise,
  Visible Mechanic). If you can't name it, the post has no
  mechanism.

## Signature treatments

- **Italic Playfair mid-sentence.** "Never Ranked. *Until now.*"
  "The blue-link era *is over.*" "Five engines. *One answer.*"
  "We built *our* expertise." Use this when the visual carries
  typography, OR when the LinkedIn post has a single line
  designed to be screenshot-quoted.
- **Receipts framing.** Engine + actual prompt + the cited domain
  underlined in gold + CITED or MISSING tag. The strongest visual
  format Never Ranked has.
- **Stat-as-hero.** "1.2%" in oversized Playfair gold. The number
  IS the post. Caption explains. Best for LinkedIn carousel slide
  one or IG square.
- **Constellation visualization.** Brand Data, Website Content,
  Knowledge Graph, Schema → Your Brand → ChatGPT, Perplexity,
  Claude, Gemini, Microsoft Copilot, Google AI Overviews, Gemma.
  Use when the post is about how AEO works mechanically.
