# Platforms — ranking #1, traffic zero

## LinkedIn — primary

- **Format:** Single image post
- **Hero:** 1200 × 630 version
- **Caption:** LinkedIn block from caption.md
- **Hashtags:** None (LinkedIn algorithm does not reward them anymore)
- **Link:** Drop `check.neverranked.com` in the caption (LinkedIn
  no longer aggressively suppresses external links the way it
  used to a year ago, but you can also drop it in the first
  comment if you want maximum reach)
- **Tag:** Tag Never Ranked company page

## X / Threads — same day

- **Format:** Single post on X. On Threads, can run as a
  2-3 reply chain if you want to expand
- **Hero:** Optional. Threads supports the same square as IG.
  X works fine without an image when the copy carries.
- **Caption:** X / Threads block from caption.md
- **Link:** End of the post

## Instagram — same day

- **Format:** Single feed post (square 1080x1080)
- **Hero:** Square version
- **Caption:** Instagram block from caption.md (with hashtags
  on their own line at the end)
- **Link:** Bio link (point at check.neverranked.com or the
  homepage that day)
- **Story:** Reshare the feed post to story with a "swipe up"
  if you have the link sticker enabled
