# Visual brief: 1.2% single-stat hero

## Concept

The hot-take stat as the entire image. One massive number, two
short framing lines above and below, italic gold mid-sentence
emphasis on the kicker. Pure typographic editorial card.

## Layout (1080 × 1350 portrait, 4:5)

**Eyebrow (top, two lines, Barlow Condensed ALL CAPS gold,
.18em letter-spacing, ~26px):**

```
WHEN AN AI OVERVIEW APPEARS
ABOVE THE BLUE LINKS
```

**Hero number (centered, Playfair Display gold #e8c767, italic,
~380px):**

```
1.2%
```

**Sub-caption (DM Mono cream, two lines, ~30px):**

```
is the click-through rate
to the sites below it.
```

**Hairline gold rule** (full-width inside 120px margin)

**Kicker (Playfair Display, three lines):**

```
Most marketers are still       (cream regular ~42px)
optimizing for                  (cream regular ~42px)
the other 98.8.                 (gold italic ~46px)
```

**Wordmark (lower right, Barlow Condensed ALL CAPS gold tracked,
~20px):**

```
NEVERRANKED.COM
```

## Visual rules

- Background: deep charcoal #121212 with subtle film grain
  overlay and corner-vignette darkening
- Gold #e8c767 used sparingly, gold italic Playfair carries the
  brand signature
- Cream #fbf8ef for the primary type that is not gold
- Hairline gold rule for division, never heavy borders
- Generous vertical negative space top and bottom
- The "1.2%" should dominate the canvas. Everything else
  supports it.

## Generation

Generated locally via Pillow at /Users/lanceroylo/Desktop/neverranked
using brand fonts (Playfair Display, DM Mono, Barlow Condensed)
downloaded from Google Fonts GitHub. Pixel-perfect placement at
exact 1080 × 1350. No Stitch round trip needed for typographic-
only graphics.

## Reusable as a template

This same composition works for any single-stat post: swap the
number, the eyebrow, the sub-caption, and the kicker. The visual
DNA stays. Future single-stat posts (e.g. "1.2 percent click-
through," "45 to 95 in 24 hours," "98 percent of buyers...") can
reuse the Pillow script with parameter changes.
