09 One-line health check

Quick way to verify the site is up across all surfaces. Paste into a terminal:

for u in / /agencies/ /security/ /case-studies/montaic/ ; do
  echo -n "neverranked.com$u "
  curl -sL -o /dev/null -w "%{http_code}\n" "https://www.neverranked.com$u"
done
curl -sL -o /dev/null -w "check.neverranked.com %{http_code}\n" "https://check.neverranked.com/"
curl -sL -o /dev/null -w "app.neverranked.com %{http_code}\n" "https://app.neverranked.com/"

Expect all 200s on marketing surfaces. Checkout endpoints will 302 (redirect to Stripe) which is correct.
