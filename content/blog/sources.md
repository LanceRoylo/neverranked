# Citation Library

Reputable sources for backing factual claims in blog posts.
Organized by topic. Every post in the library should pull
citations from this file rather than inventing sources or
linking to weak sources (random Medium posts, AI-generated
content farms, etc.).

If a claim cannot be backed by a source here, do not use it.
Or find a fresh source, vet it, and add it to this file.

---

## How to vet a new source

A source belongs in this file if it meets at least four of these:

1. Author or organization is named and has a verifiable identity
2. Publication date is visible and recent (within 24 months for
   stats, longer ok for foundational research)
3. Methodology is described or linkable
4. The site is not primarily SEO content (e.g., not a "best of"
   listicle from a marketing blog)
5. Cited by other reputable sources on the same topic

When in doubt, skip the source. A weaker post with fewer claims
beats a stronger-sounding post with shaky citations.

## Sources to avoid (low credibility)

These show up in search results but should not be cited:

- Random Medium posts (unless author is independently verifiable)
- Forbes contributor articles (the contributor network, not Forbes
  staff articles)
- Entrepreneur.com listicles
- "Top 10 X for Y" SEO content farms
- Quora answers
- Reddit threads as the only source (ok as supporting color, never
  the primary claim)
- LinkedIn articles by random authors (verify the author first)
- AI-generated SEO blog content (you can usually tell: generic
  examples, no specifics, "in today's digital landscape" type
  language)

If a fact only shows up on these kinds of sources, treat the fact
as unconfirmed and either find a real source or omit it.
